# Pacote de agentes do StoryFlame

Este pacote contém:

- `AGENTS.md` raiz com a coordenação geral
- `backend/AGENTS.md`
- `frontend/AGENTS.md`
- `ux/AGENTS.md`
- `docs/ARCHITECTURE.md`

## Como usar
1. Copie os arquivos para a raiz do repositório do StoryFlame.
2. Ajuste nomes de pastas se o seu projeto usar outra estrutura.
3. Deixe o `AGENTS.md` principal na raiz.
4. Mantenha os demais `AGENTS.md` próximos das áreas que cada agente deve seguir.

## Estrutura sugerida do projeto
```text
StoryFlame/
├─ AGENTS.md
├─ backend/
│  └─ AGENTS.md
├─ frontend/
│  └─ AGENTS.md
├─ ux/
│  └─ AGENTS.md
├─ docs/
│  ├─ ARCHITECTURE.md
│  └─ README_PACOTE.md
└─ src/
```

## Observação
Se o seu código-fonte real estiver em `src/main/java/...`, você pode manter estes arquivos de instrução fora do `src` e usá-los apenas como orientação para o Codex.
